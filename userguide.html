Languages:

Step-by-Step Instructions

  • Getting Started:
    • Download and install the app from the App Store or Google Play.
    • Open the app and create an account or log in.
    • Connect your robots by following the on-screen instructions.
    • Navigate to the Main Dashboard to see all connected robots.
  • Controlling Your Robots:
    • Select a robot from the Main Dashboard.
    • Use the Control Panel to issue commands.
    • Monitor robot status in real-time through the app.
  • Customizing Settings:
    • Go to Settings from the Main Dashboard.
    • Adjust preferences such as notifications, robot names, and control sensitivity.
    • Save changes to apply your custom settings.

Troubleshooting

  • Robot Not Connecting:
    • Ensure Bluetooth/Wi-Fi is enabled.
    • Restart the app and try connecting again.
  • App Crashes:
    • Update the app to the latest version.
    • Restart your device and reopen the app.
  • Commands Not Executing:
    • Check the robot’s battery level.
    • Ensure the robot is within range.

Software Updates

  • Enable Automatic Updates:
    • Go to your device settings and enable automatic updates for the app.

  • Check for Updates Manually:
    • Open the app store and check for available updates.
    • Download and install any updates.
  • Install Updates Promptly:
